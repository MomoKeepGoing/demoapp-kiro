# Product Overview

LinkUp (联络) is a modern instant messaging application with WhatsApp-style design, built on AWS Amplify Gen2.

## Current Features
- User authentication with email verification
- Profile management (username, avatar)
- WhatsApp-inspired UI with responsive design
- Real-time upload progress tracking
- Toast notification system

## In Development
- Real-time chat functionality
- One-on-one messaging
- Group chat
- Message notifications

## Target Language
The application UI is in Chinese (Simplified). All user-facing text, error messages, and documentation should be in Chinese unless otherwise specified.
